\doxysubsection{Detailed Description}
Base global planner using 2d occupancy maps. 

This provides an abstract interface wrapping around an occupancy grid data. Planner interfaces inheriting this base class will define ths {\ttfamily compute\+\_\+plan} function tht computes path from a 2d start point to a goal point
